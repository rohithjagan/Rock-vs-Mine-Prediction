A **Machine Learning classification project** that predicts whether a detected underwater object is a **Rock** or a **Mine (explosive)** using sonar signal data.  
This project applies supervised learning techniques to analyze frequency patterns and make accurate binary predictions.

---

## Overview

The **Sonar (Mines vs Rocks)** dataset contains 60 numerical features representing sonar signal strengths bounced off metal cylinders and rocks.  
The objective is to build a model that can distinguish between **mines** (metal objects) and **rocks** (natural objects) based on these reflected sonar signals.

---

## Features

Data preprocessing and cleaning  
Feature scaling and normalization  
Exploratory Data Analysis (EDA) with visualizations  
Model training using multiple algorithms  
Evaluation using accuracy, confusion matrix, and classification report  
User input interface for real-time prediction

## Dataset

- **Name:** Sonar, Mines vs. Rocks  
- **Source:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/connectionist+bench+(sonar,+mines+vs.+rocks))  
- **Description:**  
  - 60 features representing sonar signal energy at different angles  
  - 208 samples (104 mines, 104 rocks)  
  - **Target:** `M` (Mine) or `R` (Rock)
